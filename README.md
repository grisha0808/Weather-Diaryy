## Описание программы

Weather Diary — это графическое приложение для ведения личного дневника погоды.  
Программа позволяет:
- Добавлять записи о погоде (дата, температура, описание, наличие осадков)
- Просматривать все записи в удобной таблице
- Фильтровать записи по дате или по температуре
- Сохранять данные в JSON-файл и загружать их при следующем запуске

Приложение написано на Python с использованием библиотеки tkinter.

---

## Требования для запуска

- Python 3.6 или выше
- Библиотека tkinter (входит в стандартную поставку Python)
