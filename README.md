## Описание
Приложение для управления заметками с фильтрацией, валидацией, сохранением в JSON и функцией "Случайная заметка".

## Функции
- Добавление заметки (заголовок, текст, категория, приоритет)
- Валидация: непустой заголовок, текст ≥10 символов
- Фильтрация по категории и приоритету
- Сохранение и загрузка из notes.json
- Удаление заметок
- Случайная заметка с использованием random

## Требования
- Python 3.6+
- Стандартные библиотеки: tkinter, json, unittest, random, os
